# Cht Mod Program Schedule
The "Cht Mod Program Schedule" is a sample application created to show how to do web crawler periodically. The project is based on the [Express](https://expressjs.com/) framework and [Bootstrap](http://getbootstrap.com/) to build a simple app that is deployed to [AWS Elastic Beanstalk](http://aws.amazon.com/elasticbeanstalk/)

![screenshot](screenshot.PNG)

## Requirements:
* express
* cheerio
* node-cron

## Installation
Execute this command to install the project:
```js
npm install
```

## Run
Execute this command to run the project:
```js
node app
```
